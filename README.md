# Snake Game (Fixed Length)

## Description
A simple Snake game where the snake moves around, eats food and does grow in size.  
Built with **Python** and **Pygame**.

## Features
- Classic Snake gameplay
- Snake eats food
- Arrow key controls
- Score tracking

## Requirements
- Python 3.x
- Pygame
